## Installation
### Git
`ln -s dotfiles/gitconfig_global .gitconfig`
### Bashrc
`ln -s ~/dotfiles/.bashrc .bashrc` or append `. ~/dotfiles/.bashrc` to .bashrc
### TMUX
`ln -s dotfiles/.tmux.conf .tmux.conf`
### VIM
`ln -s dotfiles/.vim .vim`
`ln -s dotfiles/.vimrc .vimrc`
