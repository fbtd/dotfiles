## Installation
### Git
`ln -s ~/dotfiles/gitconfig_global .gitconfig`
### Bashrc
remove .bashrc and
`ln -s ~/dotfiles/.bashrc .bashrc`<br/>
or append `. ~/dotfiles/.bashrc` to .bashrc
### TMUX
`ln -s dotfiles/.tmux.conf .tmux.conf`
### VIM
`ln -s dotfiles/.vim .vim`<br/>
`ln -s dotfiles/.vimrc .vimrc`
